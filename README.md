# CoLab - Collaborative Learning Platform

## Overview

CoLab is a learning platform that helps students access course materials, collaborate with classmates, and manage their learning experience in one place.

## Features

- **Profile & Avatar**: Create a personalized profile with your chosen avatar
- **Lectures & Tutorials**: Access course materials and learning resources
- **Team Collaboration**: Connect and team up with classmates for group projects
- **Dark/Light Theme**: Switch between dark and light themes as needed
- **Mobile Friendly**: Works smoothly on phones, tablets, and computers
- **Quick Access**: Simple navigation to all course materials and tools

## How to Use

1. Start at the home page
2. Create your profile by selecting an avatar
3. Browse and access lectures and tutorials
4. Use TeamUp to find and connect with teammates
5. Log out when finished

## Pages

- **Home**: Main landing page for students
- **Lectures**: Access video lectures and course content
- **Tutorials**: Interactive tutorials for hands-on learning
- **TeamUp**: Find teammates and collaborate on projects
- **Quiz**: Take quizzes to test your knowledge
- **Profile**: Manage your profile and settings

## Requirements

- A modern web browser (Chrome, Firefox, Safari, Edge)
- JavaScript enabled
- No installation needed
